## Atividade: Desempenho Acadêmico (Base do Projeto)

Este repositório contém a solução da atividade de desempenho acadêmico
usando listas, tuplas e um relatório simples em arquivo `.txt`.

Nesta base inicial temos apenas:
- um arquivo `main.py` vazio (apenas estrutura);
- um arquivo `processamento.py` vazio (será preenchido aos poucos);
- este `README` com o contexto geral.

As funcionalidades serão implementadas **aos poucos**, em branches separadas,
seguindo a ideia de desenvolvimento incremental (Kanban / Git Flow simples).
